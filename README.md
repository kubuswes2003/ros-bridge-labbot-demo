# ROS1-ROS2 Bridge Demo for Labbot Robot

Projekt demonstracyjny komunikacji między ROS1 a ROS2 using ros1_bridge.

## Działająca wersja (v1.0-working)
- ROS1: Noetic 
- ROS2: Foxy
- Bridge: dynamic_bridge --bridge-all-topics

## Uruchomienie:
```bash
docker-compose up
```
